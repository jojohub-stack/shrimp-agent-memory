## 🧠 知識圖譜（Ontology）

### 實體類型
- **Person**: Rene、家人、朋友
- **Project**: 網格交易、P.A.L.、Agent Office
- **Task**: 待辦事項、定期任務
- **Event**: 會議、截止日期
- **Document**: 報告、筆記
- **Stock**: 投資標的（眾達-KY、聯鈞...）
- **Crypto**: 加密貨幣（SOL、FET、PEPE...）

### 關係類型
- `WORKS_ON` → Rene 工作於 專案
- `INVESTS_IN` → Rene 投資 股票/加密貨幣
- `RELATED_TO` → 文件 相關 專案
- `DEPENDS_ON` → 任務 依賴 任務
- `LEARNED_FROM` → 教訓 來自 失敗

## 🔄 自動學習流程

```
每日流程：
1. 任務執行 → 記錄到 memory/YYYY-MM-DD.md
2. 遇到失敗 → 立即記錄教訓
3. 23:00 → 執行每日反思
4. 反思結果 → 更新 Memoria

每週流程：
1. 週日 20:00 → 執行每週回顧
2. 整理 daily notes → 提煉到 MEMORY.md
3. 更新 Ontology 圖譜
4. 評估技能樹需求

每月流程：
1. 月末 → 深度回顧
2. 分析成功/失敗模式
3. 規劃下月改進目標
```
