# 伊朗戰爭與石油危機對台灣股市影響報告

**報告日期：** 2026年3月13日
**資料來源：** CNN, IEA, 國際能源機構

---

## 一、戰爭現況概要

### 1.1 衝突背景

- **開戰日期：** 2026年2月28日
- **戰爭天數：** 第13天（截至3月12日）
- **交戰方：** 美國 + 以色列 vs 伊朗
- **伊朗新最高領袖：** Mojtaba Khamenei（接替被擊殺的前領袖）

### 1.2 戰爭關鍵進展

| 日期 | 重大事件 |
|------|----------|
| 2/28 | 美以聯合空襲伊朗，戰爭爆發 |
| 3/2 | 美國 THAAD 雷達基地在約旦、沙烏地阿拉伯、阿聯酋遭攻擊 |
| 3/5 | 伊朗開始在霍爾木茲海峽佈雷 |
| 3/8-11 | 六艘船隻在波斯灣遭攻擊 |
| 3/12 | 伊朗新最高領袖發布首個聲明，誓言關閉霍爾木茲海峽 |

### 1.3 軍事動態

- 伊朗擁有 **5,000-6,000 枚海軍水雷**
- IRGC（伊斯蘭革命衛隊）小型艦隊仍保持 **80-90% 完整**
- 俄羅斯正向伊朗提供無人機戰術建議
- 伊朗威脅攻擊中東所有美軍基地

## 二、石油危機分析

### 2.1 霍爾木茲海峽的重要性

```
┌─────────────────────────────────────────────────────┐
│           霍爾木茲海峽戰略地位                       │
├─────────────────────────────────────────────────────┤
│  • 全球 20% 石油供應經此通道                         │
│  • 每日約 2,100 萬桶原油通過                         │
│  • 最窄處僅 21 英里寬                               │
│  • 連接波斯灣與阿拉伯海的唯一出口                    │
│  • 全球能源供應的「咽喉要道」                        │
└─────────────────────────────────────────────────────┘
```

### 2.2 油價走勢

| 指標 | 戰前 | 目前 | 漲幅 |
|------|------|------|------|
| **布倫特原油** | ~$75-80 | **$100.46** | +9.22% |
| **WTI 原油** | ~$70-75 | **$95.73** | +9.72% |
| **週一高點** | - | ~$120 | 峰值 |

> **重要里程碑：** 布倫特原油自 2022年8月以來首次收於 $100 以上

### 2.3 國際應對措施

- **IEA 釋出戰備儲油：** 4 億桶
- **美國能源部長表態：** 霍爾木茲海峽「必須也將會」重新開放
- **IEA 評估：** 這是「全球石油市場史上最大的供應中斷」

## 三、全球股市影響

### 3.1 美股表現（3/12）

| 指數 | 跌幅 | 連跌天數 |
|------|------|----------|
| 道瓊工業 | -739點 (-1.56%) | 3天 |
| S&P 500 | -1.52% | 3天 |
| 納斯達克 | -1.78% | - |

### 3.2 債市反應

- **10年期美債收益率：** 升至 4.26%（2月初以來最高）
- **原因：** 投資人重新評估能源價格上漲帶來的通膨風險

### 3.3 市場情緒

- **恐懼指數：** 顯著上升
- **資金流向：** 避險資產（黃金、美元）受追捧
- **分析師觀點：** 霍爾木茲海峽是「全球市場的關鍵樞紐」

## 四、對台灣股市的潛在影響

### 4.1 直接影響因素

| 影響面向 | 說明 | 影響程度 |
|----------|------|----------|
| **能源成本** | 台灣 98% 能源依賴進口 | ⚠️ 高 |
| **通膨壓力** | 油價上漲推升物價 | ⚠️ 高 |
| **半導體供應鏈** | 運輸成本上升、延遲 | ⚠️ 中高 |
| **電子業毛利** | 能源成本侵蝕利潤 | ⚠️ 中 |
| **航運股** | 短期利多（運價上漲） | ✅ 正面 |

### 4.2 產業別影響評估

#### 🔴 負面影響產業

| 產業 | 原因 | 預期影響 |
|------|------|----------|
| **航空業** | 燃油成本暴增 | 獲利大幅下滑 |
| **塑化業** | 原料成本上升 | 毛利受壓縮 |
| **汽車業** | 銷售成本增加 | 需求可能下滑 |
| **水泥/鋼鐵** | 生產能源成本上升 | 獲利受影響 |
| **食品業** | 運輸成本、原物料上漲 | 漲價壓力 |

#### 🟢 正面影響產業

| 產業 | 原因 | 預期影響 |
|------|------|----------|
| **航運業** | 運價上漲、繞行需求 | 營收增加 |
| **綠能/電動車** | 替代能源需求加速 | 長期利多 |
| **國防股** | 軍事衝突升溫 | 訂單可能增加 |
| **太陽能/風電** | 能源自主需求 | 政策支持加強 |

#### ⚪ 中性/待觀察

| 產業 | 原因 | 建議 |
|------|------|------|
| **半導體** | 需求穩定但成本上升 | 觀察終端需求 |
| **金融業** | 升息環境有利，但風險增加 | 謹慎佈局 |
| **台積電** | 議價能力強，可轉嫁成本 | 長期仍看好 |

### 4.3 歷史參考案例

| 事件 | 油價漲幅 | 台股表現 |
|------|----------|----------|
| 1990 波斯灣戰爭 | +100% | 大跌後反彈 |
| 2008 油價高峰 | 至$147 | 金融海嘯重挫 |
| 2022 俄烏戰爭 | 至$120 | 短期震盪後回穩 |

## 五、投資策略建議

### 5.1 短期策略（1-3個月）

```
⚠️ 風險規避
├── 減碼：航空、汽車、高耗能產業
├── 加碼：航運、綠能、國防
└── 持有：半導體龍頭、高股息防禦股
```

### 5.2 中期策略（3-12個月）

- **關注戰爭走向：** 霍爾木茲海峽是否重新開放是關鍵
- **油價回落時機：** 若戰事緩和，可佈局受壓產業
- **政策受益股：** 台灣能源轉型政策加速下的受惠者

### 5.3 關鍵觀察指標

| 指標 | 意義 | 關注點 |
|------|------|--------|
| **布倫特原油** | 能源成本基準 | 是否突破 $120 |
| **霍爾木茲海峽** | 供應瓶頸 | 是否重新開放 |
| **美元指數** | 資金流向 | 強勢美元不利新興市場 |
| **VIX 恐懼指數** | 市場情緒 | >30 表示恐慌 |
| **台幣匯率** | 資金動向 | 貶值可能加速外資流出 |

## 六、結論

### 6.1 核心判斷

**戰爭是最大的不確定因素。** 霍爾木茹海峽的封鎖已造成「全球石油市場史上最大供應中斷」，油價重回 $100 是一個重要信號。

### 6.2 對台股影響總結

| 時間框架 | 影響評估 |
|----------|----------|
| **短期** | 震盪加大，能源相關股波動劇烈 |
| **中期** | 取決於戰爭持續時間，若超過3個月將明顯影響經濟 |
| **長期** | 加速台灣能源轉型，綠能產業受惠 |

### 6.3 投資人建議

1. **保持現金部位：** 應對市場波動
2. **分散風險：** 避免過度集中單一產業
3. **關注戰況：** 霍爾木茹海峽是關鍵轉折點
4. **長期佈局：** 綠能、電動車、半導體龍頭
