## 🔒 安全防護系統

### AI Agent OPSEC（已啟用）
- **位置：** `skills/ai-agent-opsec/`
- **功能：** 防止機密資料外洩到外部 API
- **中間件：** 所有記憶系統輸入輸出都經過安全掃描
- **分級：**
  - 外部服務（web_search, telegram）→ 完全禁止機密資料
  - 內部服務（memory, ontology）→ 防禦性編輯

### 機密詞彙完整性監控（已啟用）
- **目標：** `classified/classified-terms.md`
- **頻率：** 每小時整點（cron）
- **觸發：** SHA256 雜湊值變化時發送 Telegram 警報
- **腳本：** `scripts/file-integrity-monitor.sh`

### 危險操作確認（已啟用）
- **檢測：** `rm`, `transfer`, `withdraw`, 私鑰等
- **白名單：** 刪除 `.log`, `.tmp`, `data/*.json` 不需確認
- **流程：** 危險操作 → Telegram 確認 → 執行

### Skill Vetting 流程（強制執行）
- **規則：** 安裝任何 skill 之前必須先審計
- **腳本：** `bash ~/workspace/scripts/vet_skill.sh <skill_name>`
- **檢查項目：**

  - 🚨 發送數據到外部
  - 🚨 請求憑證

- **同步位置：** secrets.env → .bashrc → crontab
- **快捷指令：** `think`（查看所有環境變量）
