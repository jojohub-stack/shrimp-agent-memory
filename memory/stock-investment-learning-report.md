## 一、股票投資分析方法

### 1.1 基本面分析

**核心觀念：** 分析公司內在價值，尋找股價低於價值的股票

**四大構面：**

| 構面 | 重點 | 關鍵指標 |
|------|------|----------|
| **財務報表** | 損益表、資產負債表、現金流量表 | 營收、淨利、現金流 |
| **獲利能力** | 公司賺錢能力 | ROE、ROA、毛利率、淨利率 |
| **成長性** | 持續成長動能 | 營收成長率、EPS成長率 |
| **估值評價** | 股價是否合理 | 本益比(P/E)、股價淨值比(P/B) |

**關鍵財務比率：**

| 指標 | 公式 | 判斷標準 |
|------|------|----------|
| **本益比(P/E)** | 股價/EPS | <15 偏低，15-25 合理，>25 偏高 |
| **股價淨值比(P/B)** | 股價/每股淨值 | <1 被低估，1-2 合理 |
| **ROE** | 淨值報酬率 | >15% 優秀，>20% 極佳 |
| **股息殖利率** | 每股股利/股價 | 3-6% 合理，>6% 高殖利率 |

### 1.2 技術面分析

**核心觀念：** 透過價量關係預測股價走勢

**K線形態：**

| 形態 | 意義 | 後續走勢 |
|------|------|----------|
| 長紅 K 線 | 買方強勢 | 上漲機率高 |
| 長黑 K 線 | 賣方強勢 | 下跌機率高 |
| 十字線 | 多空均衡 | 變盤訊號 |
| 鎚子線 | 反轉訊號 | 底部反轉 |
| 流星線 | 反轉訊號 | 頭部反轉 |

**均線系統：**

| 均線 | 意義 | 用途 |
|------|------|------|
| 5 日線 | 短期趨勢 | 短線進出 |
| 20 日線 | 月線 | 中短期趨勢 |
| 60 日線 | 季線 | 中期趨勢 |
| 120 日線 | 半年線 | 中長期趨勢 |
| 240 日線 | 年線 | 長期趨勢 |

**技術指標：**

| 指標 | 計算原理 | 應用方式 |
|------|----------|----------|
| **KD 值** | 隨機指標 | K>80 超買，K<20 超賣 |
| **RSI** | 相對強弱指標 | >70 超買，<30 超賣 |
| **MACD** | 指數平滑異同移動平均線 | 黃金交叉買進，死亡交叉賣出 |
| **成交量** | 量先價行 | 價漲量增健康，價漲量縮背離 |

## 二、股票預測方法

### 2.1 傳統預測方法

**趨勢分析：**
- 均線排列：多頭排列（短>中>長）、空頭排列
- 趨勢線：連接低點形成上升趨勢線
- 形態學：頭肩頂/底、M頭、W底、三角形整理

**景氣週期：**

| 階段 | 特徵 | 投資策略 |
|------|------|----------|
| 復甦期 | 景氣燈號藍轉綠 | 開始佈局 |
| 繁榮期 | 景氣燈號綠轉紅 | 持股續抱 |
| 衰退期 | 景氣燈號紅轉綠 | 獲利了結 |
| 谷底期 | 景氣燈號藍燈 | 分批進場 |

**2026 年景氣對策信號：**
- 2026年1月綜合判斷分數 39 分，連續亮紅燈
- 景氣熱絡，但需注意過熱風險
- 策略：獲利了結部分持股

### 2.2 AI/機器學習預測

**量化模型：**
- 迴歸分析：預測股價走勢
- 時間序列：ARIMA、LSTM
- 機器學習：隨機森林、XGBoost
- 深度學習：神經網絡預測

**AI 預測限制：**
- 歷史數據無法完全預測未來
- 市場情緒難以量化
- 黑天鵝事件無法預測
- 準確率約 55-65%

## 三、選股策略

### 3.1 價值投資

**核心理念：** 買入被低估的優質公司，長期持有

**選股標準：**

| 條件 | 標準 | 說明 |
|------|------|------|
| 本益比 | < 15 | 股價相對便宜 |
| 股價淨值比 | < 1.5 | 低於淨值 |
| ROE | > 15% | 獲利能力佳 |
| 股息殖利率 | > 3% | 穩定配息 |
| 負債比 | < 50% | 財務穩健 |

**適合族群：** 穩健型投資人、長期投資者

### 3.2 成長投資

**核心理念：** 買入高成長公司，享受成長果實

**選股標準：**

| 條件 | 標準 | 說明 |
|------|------|------|
| 營收成長率 | > 20% | 高成長 |
| EPS 成長率 | > 15% | 獲利成長 |
| 毛利率 | > 30% | 產品競爭力 |
| 市佔率 | 持續提升 | 產業地位 |

**適合族群：** 積極型投資人、中期投資者

### 3.3 動能投資

**核心理念：** 追逐強勢股，顺势而為

**選股標準：**

| 條件 | 標準 |
|------|------|
| 價格突破 | 突破整理區間 |
| 量價關係 | 價漲量增 |
| 均線排列 | 多頭排列 |
| 技術指標 | KD、MACD 多頭訊號 |

**適合族群：** 短線交易者

### 3.4 股息投資

**核心理念：** 穩定領息，複利成長

**選股標準：**

| 條件 | 標準 |
|------|------|
| 殖利率 | > 5% |
| 配息穩定 | 連續配息 5 年以上 |
| 填息率 | 歷史填息率高 |
| 本益比 | < 20 |

**適合族群：** 保守型投資人、退休族

## 四、風險管理

### 4.1 資金管理

**基本原則：**

| 原則 | 說明 |
|------|------|
| **單一標的** | 不超過總資金 10-20% |
| **產業分散** | 不超過總資金 30% |
| **保留現金** | 保留 10-20% 現金 |
| **分批進出** | 避免一次性進出 |

### 4.2 停損停利

**停損策略：**

| 方法 | 說明 |
|------|------|
| **固定比例** | 跌破買進價 -7% 停損 |
| **技術停損** | 跌破支撐、均線 |
| **時間停損** | 買進後 N 天無表現 |

**停利策略：**

| 方法 | 說明 |
|------|------|
| **固定比例** | 上漲 +15-20% 部分停利 |
| **技術停利** | 破壞上漲趨勢線 |
| **目標價** | 達到合理價位 |

### 4.3 資產配置

**2026 建議配置：**

| 資產類別 | 比例 | 說明 |
|----------|------|------|
| **股票** | 40-60% | 成長動能 |
| **債券** | 20-30% | 防禦配置 |
| **現金** | 10-20% | 彈性運用 |
| **另類投資** | 5-10% | 黃金、加密貨幣 |
