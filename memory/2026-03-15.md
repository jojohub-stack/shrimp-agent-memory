## Web3 + Notion 整合

### 安裝
- ethers.js v6.16.0 (Node.js)
- web3.py v7.14.1 (Python 虛擬環境)

### 工具
`scripts/web3_notion.js` - 區塊鏈數據查詢 + Notion 記錄

### 功能
- `balance <address>` - 查詢 ETH 餘額
- `token <address> <token>` - 查詢 ERC20 代幣
- `tokens <address>` - 查詢所有常用代幣
- `gas` - 查詢 Gas 價格
- `block` - 查詢最新區塊
- `wallet <address>` - 錢包總覽

### 支援網路
- ethereum, polygon, arbitrum, optimism, bsc

### 代幣
- USDT, USDC, DAI, WETH, LINK, UNI

### 範例
```bash
# 查詢餘額並存入 Notion
dotenvx run -- node scripts/web3_notion.js balance 0x... --save

# 查詢 Gas
dotenvx run -- node scripts/web3_notion.js gas ethereum

# 錢包總覽
dotenvx run -- node scripts/web3_notion.js wallet 0x... --save
```

### 安全設定
- 使用 dotenvx 加密 `.env` 檔案
- 私鑰存於 `.env.keys`（不進 Git）
- `DOTENV_PRIVATE_KEY` 已加入 `~/.bashrc`
- **本地加密庫**：`/root/.openclaw/workspace/.vault/secrets.enc`

### 本地加密工具

## 動態網格交易系統

### 目錄
`/root/.openclaw/workspace/skills/grid-trader/`

### 功能
- 📊 市場監控（價格、MA、ATR、RSI、趨勢）
- 🔧 網格策略（自動計算區間、掛單）
- 🤖 動態調整（價格突破自動移動區間）
- 🧪 紙上交易（模擬模式，不動用真實資金）
- 📝 Notion 記錄
- 🔔 Telegram 通知

### 使用方式
```bash
cd /root/.openclaw/workspace/skills/grid-trader

# 查看狀態
python3 scripts/grid_bot.py --status

# 啟動紙上交易（模擬）
python3 scripts/grid_bot.py --mode paper

# 啟動監控模式
python3 scripts/grid_bot.py --mode monitor

# 自訂參數
python3 scripts/grid_bot.py --symbol bitcoin --interval 300
```

### 配置
編輯 `config.json`：
```json
{
  "grid_count": 20,        // 網格數量
  "range_pct": 0.05,       // 區間百分比 (5%)
  "position_size": 100,    // 每格金額
  "check_interval": 300    // 檢查間隔（秒）
}
```

### 模式說明
| 模式 | 說明 |
|------|------|
| paper | 紙上交易（模擬） |
| live | 實盤交易（需 API） |
| monitor | 只監控不交易 |
