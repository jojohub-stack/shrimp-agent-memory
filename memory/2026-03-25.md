## 🕒 19:30 Social-Monitor 翻譯修復

### 問題

Social-monitor 新聞翻譯功能一直沒生效，英文標題沒有被翻譯成繁體中文。

### 根本原因

響應格式解析錯誤：
- **CLI Proxy 返回**: OpenAI 格式 `choices[0].message.content`
- **social-monitor 解析**: Google 格式 `candidates[0].content.parts[0].text`

### 修復

修改 `skills/social-monitor/scripts/monitor.py` 的 `translate_to_chinese()` 函數：

```python
# OpenAI 格式
choices = result.get('choices', [])
if choices:
    content = choices[0].get('message', {}).get('content', '')
    if content:
        return content.strip()[:500]
# 備用：Google 格式
candidates = result.get('candidates', [])
...
```

### 測試結果

| 英文 | 繁中 |
|------|------|
| Bitcoin hits new all-time high | 比特幣創下新歷史最高價值 |
| Ethereum price surges 10% | 以太坊價格暴漲10% |
