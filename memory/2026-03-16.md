
**使用方式：**
```bash

python3 scripts/grid_bot.py --mode paper

# 運行單一交易對（獨立資金）
python3 scripts/grid_bot.py --mode paper --symbol solana

# 查看狀態
python3 scripts/grid_bot.py --status
```

**狀態顯示：**
- 各交易對：價格、趨勢、網格區間、成交數
- 共用帳戶：初始資金、權益、餘額、持倉、總盈虧
- **合併報告**：所有交易對整合成一則通知

### 安全中間件整合

#### AI Agent OPSEC 設為 Middleware
- **位置：** `skills/ai-agent-opsec/security_middleware.py`
- **功能：** 所有記憶系統輸入輸出都經過安全掃描

**中間件架構：**
```
┌─────────────────────────────────────────────────────────┐
│                    Security Middleware                    │
├─────────────────────────────────────────────────────────┤
│  All Inputs ────→ gate_input() ────→ Sanitized          │
│  All Outputs ───→ gate_output() ───→ Sanitized          │
│       ├── External (web, telegram) → BLOCKED            │
│       └── Internal (memory, files) → REDACTED            │
│  Subagent Spawn → gate_before_spawn() → Safe Task       │
└─────────────────────────────────────────────────────────┘
```

**服務分級：**
- **外部服務：** web_search, telegram, email, http → 完全禁止機密資料
- **內部服務：** memory, ontology, file_write → 防禦性編輯

### 環境變量統一管理

#### 管理腳本
```bash
~/workspace/scripts/manage_secrets.sh
```

#### 使用方式
```bash
# 新增環境變量
./manage_secrets.sh add NEW_API_KEY "your-key-here"

# 更新環境變量
./manage_secrets.sh update DASHSCOPE_API_KEY "new-key"

# 移除環境變量
./manage_secrets.sh remove OLD_KEY

# 列出所有環境變量
./manage_secrets.sh list

# 強制同步所有位置
./manage_secrets.sh sync
```

#### 自動同步位置
1. `/root/.openclaw/secrets.env` (systemd)
2. `~/.bashrc` (終端)
3. `crontab` (定時任務)

#### 相關檔案
- `scripts/manage_secrets.sh` - 統一管理腳本
- `.openclaw/secrets.env` - 主存儲文件

#### 快捷指令（終端別名）
```bash
# 查看所有環境變量
think

# 同步並重啟 OpenClaw
refresh
```

#### 環境變量清單（9 個）
```
OPENROUTER_API_KEY    - OpenRouter API
DASHSCOPE_API_KEY     - 百鍊/阿里雲
TELEGRAM_BOT_TOKEN    - Telegram 機器人
TELEGRAM_CHAT_ID      - Telegram Chat ID
NVIDIA_API_KEY        - NVIDIA NIM
GITHUB_API_KEY        - GitHub Models
MISTRAL_API_KEY       - Mistral AI
GROQ_API_KEY          - Groq
TAVILY_API_KEY        - Tavily 搜尋
```

#### 架構說明
**OpenClaw 不支援環境變量引用，必須維護兩處：**

1. **openclaw.json**（硬編碼）→ OpenClaw 服務
2. **secrets.env**（9 個變量）→ Python / 終端 / cron

**更新 API keys 時，我會同步兩邊。**

#### crontab 方式
每個 cron 任務執行前自動 source：
```bash
. /root/.openclaw/secrets.env && /path/to/script.py
```
