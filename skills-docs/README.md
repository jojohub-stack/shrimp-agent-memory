## 🚀 快速開始

### 安裝技能

```bash
# 使用 clawhub 安裝
clawhub install <skill-name>

# 或手動複製
cp -r /path/to/skill ~/.openclaw/workspace/skills/
```

### 查看技能文檔

```bash
cat skills/<skill-name>/SKILL.md
```
